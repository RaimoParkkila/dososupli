# DosoSupli

React Native + Expo + SQLite supplement tracking app.

## Features
- Add supplements (name, purpose, dose)
- Calendar planning system
- Mark supplements as taken (YES / NOT toggle)
- Delete entries
- Local offline database (SQLite)

## Tech Stack
- React Native
- Expo
- SQLite (expo-sqlite)

## Status
MVP working version (in progress)
